Написать приложение для учета кошек

- добавлять
- находить по параметру (id, имя ...)
- изменять данные
- удалять

CRUD
- create
- read
- update
- delete

============================================

хранение данных :
Cat
- id
- name


repository (отдельный класс в котором будет создана коллекция и будут методы,
которые позволят совершать все необходимые действия с данными CRUD

CatRepository
- List <Cat> cats;

- необходимые методы:
 - public Cat add(String name)
 ...

 services - классы которые будут содержать методы с расширенной бизнес-логикой,
 которые будут обращаться к методам репозитория, передавать туда нужные данные
 и получать результат

 например: проверка корректности данных которые сервис получает от пользователя
